145 BPM speed garage like Interplanetary Criminal / Y U QT.

Kick:
- Use a fat 909-style / tech-house-style kick.
- Tune it to the bassline / key if needed.

Bassline writing:
- Key center here is F minor.
- Build the line from only three scale degrees / functions: root, fourth, sixth.
- Repeat those notes across different octaves.
- The groove comes from the rhythm and the octave jumps, especially the 16th-note lift into higher notes.

Bass sound:
- Use Operator.
- Build the bass from three sine waves at different octaves.
- Detune one slightly.
- Create the reverse / sucking-into-the-note character with an attack envelope on one oscillator.
- Add a little FM for extra bite.
- Use a low-pass filter.
- Use Auto Pan to simulate sidechain.

Percussion and loops:
- A quiet percussion loop can add a lot of energy, especially in the mid-range and highs.
- Clap goes at the end of each bar.
- Main hi-hat is extremely short, almost just a 16th-note tick.
- High-passed sidechained loops are crucial to the classic speed-garage sound.

Vocal atmosphere:
- Build it from one vocal sample chopped into two complementary layers.
- Pitch the source to fit the tune.
- Put the first half through reverb, sidechain, and high-pass.
- Put the second half through the same chain but add delay and pan it so the phrase opens out.
- Add a third sparse punctuation vocal for occasional accents with delay, Auto Pan, and high-pass.
