145 BPM speed garage in the style of Y U QT. Tempo lane suggested as roughly 140-150 BPM.

Main stab:
- Starts from a classic brass / orchestral stab sample.
- Pitch the stab down to the root note so it fits the bass and groove.
- Pitching it down also slows the internal sample movement, which helps it sit with the beat.
- Process with OTT to bring out the mids and highs.
- Add delay.
- Use EQ as sound-design EQ: cut lows added by OTT and aggressively boost the highs if the sound needs it.
- Sidechain the stab to the kick using a filtered detector so the ducking focuses on the top end of the kick and feels tighter.

Vocal atmosphere:
- Start from a vocal snippet.
- Stretch it and pitch it so it is in key.
- Reverb it heavily so it becomes an atmosphere layer rather than a direct vocal.
- Bringing the reverb size down can focus the wash and make it feel tighter.
- High-pass the result but leave the useful fundamental area.
- Sidechain it with the same filtered sidechain approach.

Bassline writing:
- Define a chord progression first: F minor, C minor, C# major, D# major.
- Build the bassline from the root notes of those chords.
- Use octave jumps to create swing and movement.
- Keep the notes tight and slightly separated rather than letting them fully touch.

Bass sound design:
- Use Operator.
- Start from two sine waves FM-ing each other.
- Put one oscillator an octave up and detune it a little so the melody reads more clearly.
- Low-pass the sound.
- Turn filter key tracking off so the cutoff stays fixed no matter which note is played.
- Create width with a separate return layer:
  - saturate it a little,
  - high-pass the lows,
  - add chorus,
  - high-pass again to keep the width out of the sub.
- Keep the clean low-end in the main bass and let the return carry the width.
- Sidechain both the return and the main bass to the kick.

Drums:
- Start with three short, punchy core layers: kick, clap, hi-hat.
- Kick:
  - short and punchy,
  - cut a little low end,
  - use saturator to bring it forward.
- Clap:
  - short and punchy.
- Main hi-hat:
  - very short,
  - pitched down slightly.

Background percussion:
- Use a steady extra hat plus frequent rimshots to create groove around the held bass.
- Reuse the same rimshot pitched differently on later hits to get more mileage from one sample.

Percussion loops:
- Layer multiple percussion loops that each fill different frequency roles.
- One loop can be mid-rangey, another upper-high, another in between.
- High-pass them together on the group so they do not fight the bass.
- Use Auto Pan / volume shaping to keep the loops ducked and controlled around the kick.
